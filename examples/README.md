# Examples

This folder contains example scripts demonstrating various use cases of the IDX Stock Analyzer.

## Files

### basic_usage.py
Shows how to use the analyzer programmatically from Python.

```bash
uv run python examples/basic_usage.py
```

### batch_analysis.sh
Bash script for analyzing multiple stocks and saving results.

```bash
chmod +x examples/batch_analysis.sh
./examples/batch_analysis.sh
```

### compare_stocks.py
Compares multiple stocks to find the best opportunities.

```bash
uv run python examples/compare_stocks.py
```

## Tips

- Modify the stock lists in these scripts to analyze your own portfolio
- Use these as templates for building your own analysis tools
- Combine with cron jobs for automated daily analysis
